Python Overview

Python is dynamic typed language, which means unlike c, c++, java etc, which require variable declaration with type before assignment, in python we can assign any data type to any variable and we don't even have to declare varaible before using it.

For further understanding you can check python documentation.

Data types

Baisc data types in programming language remains same, such as integer, string, float, boolean. These are the basics data types in Python. There are also advance types such at complex, bytes, set etc. You can check the built-in section to those.

Defining a variable storing integer data type. 

                
        >>> a = 1;
        >>> a
        1

Storing string type 

                
        >>> a = "Paradox";
        >>> a
        'Paradox'

As you can see, we stored integer and string in same variable, and no variable declaration is required.

Storing float type 

                
        >>> a = 10.0;
        >>> a
        10.0

If you want to store boolean type, then 

                
        >>> a = True;
        >>> a
        true

Types conversion

So for example you defined a variable, and now you want to convert it's type to another. Let's say from integer to string 

                
        >>> a = 100
        >>> a
        100
        >>> a = str(a)
        >>> a
        '100'
        >>>

As you can see we first assign 100 to a, then we use str() class to convert to convert the 100 to string type, we get output '100'. Similary using float(), int() classes, we can convert data types.

Comments

Comments are important part of programming, it's help you as well as other understand the flow of code or what a particular statement is intended to do. 

                
        a = 100

        # The value of a is 100
        print(a)

You can notice we use the # to start the comment. This is how we write comment Python.

Now if you want to write multi-line comment, one way to start each line with #. 

                
        a = 100

        # The value of a is 100,
        # then we are going to print the value.
        print(a)

Another way to write is using triple comma """ """. You have to put tiple comma at the start of comment line and at the end. 

                
        a = 100

        """ The value of a is 100,
        then we are going to print the value.
        """
        print(a)
